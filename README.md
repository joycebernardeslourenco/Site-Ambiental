# Site-Ambiental
Site/jogo sobre o meio-ambiente. Visando criar consciência na população para o mesmo. O objetivo é entender que destruindo a natureza, a preço é caro tanto para o meio ambiente como para a humanidade.

Como Jogar:
Feito no P5.Js Web Editor;
Clique no botão verde;

Use as teclas: "g" para fauna, "f" para flores, "s" para solo e "a" para água.

Dinheiro:

Lucro: 90

Meta: 500

Destruindo a natureza voce vera que seu dinheiro sobe rapido mas depois comeca a cair da mesma forma.

Existe como você curar o meio ambiente. As vezes pode certo, as vezes não.

Voce precisa cumprir a meta de 500 reais. Abaixo disso, voce perde e nao consegue estabilizar por completo a natureza.
