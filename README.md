# Site-Ambiental
Site/jogo sobre o meio-ambiente. Visando criar consciência na população para o mesmo. O objetivo é entender que destruindo a natureza, o preço é caro tanto para o meio ambiente como para a humanidade.

Como Jogar:
Feito no P5.Js Web Editor;
Clique no botão verde;

Use as teclas: "g" para fauna, "f" para flores, "s" para solo e "a" para água.

Dinheiro:

Lucro: 300

Meta: 500

Utilizando a natureza voce vera que seu dinheiro sobe rapido mas depois comeca a cair da mesma forma.

Existe como você curar o meio ambiente. Clicando em: 1 (Flora), 2 (Agua), 3 (Fauna) e 4 (Solo).

Voce precisa cumprir a meta de 500 reais. Abaixo disso, voce perde e nao consegue estabilizar por completo a natureza.
